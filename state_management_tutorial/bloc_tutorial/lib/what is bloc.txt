Bloc provider is a Widget which create and provide bloc to all of its children

Bloc is a dependency injection widget

"Bloc Provider will provide a single instance of a bloc to the subtree below it"

If we instance a bloc provider at the top, it will be available to entire subtree below it

by default bloc provider creates the bloc lazily,
meaning that the function will be executed when the bloc is look up for via bloc provider or bloc context.

bloc will be automatically closes the stream when it doesn't used.
